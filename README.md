# MANARA CSE  - C++ Programming Assignments

## 📘 Course Overview

This repository contains C++ programming assignments designed for CyberSecurity Engineers .


---

## 📂 Repository Structure

Each file represents a separate assignment:

```
Task01.cpp
Task02.cpp
...
```

Each assignment includes:

* Problem description
* Instructions
* Incomplete code for you to finish

---

## 📌 Rules

* Do NOT copy code from others
* Try to solve the problem yourself first
* Follow instructions inside each file carefully
---

## 💡 Tips

* Start with the first assignments (they are easier)
* Test your program with different inputs
* Read error messages carefully when compiling
* Ask questions if something is unclear

---

## ⚠️ Important Notes

* Use only the concepts required in each assignment
* Avoid using advanced topics unless instructed
* Make sure your program compiles without errors

---

## 🎯 Goal

By completing these assignments, you will build a strong foundation in C++ programming and problem-solving.

---

Good luck 🚀
